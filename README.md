# 🔗 Simple URL Shortener

A full-stack URL shortener built with:

- Node.js
- Express.js
- MongoDB
- React.js

---

## 🚀 Features
- Shorten long URLs
- Redirect using short code
- Pagination history
- Copy to clipboard
- Modern UI

---

## ⚙️ Installation

### Backend
npm install
npm run dev

### Frontend
npm install
npm start

---

## 📡 API Endpoints

POST /shorten  
GET /:code  
GET /history?page=1&limit=5
